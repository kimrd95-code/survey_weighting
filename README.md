# Взвешивание данных опросов

Простой веб-сервис на **Streamlit** для расчёта весов респондентов: калибровка по внешним целям (Росстат) или внутреннее взвешивание «группа на группу». В исходный Excel добавляется колонка **WT** (средний вес по выборке = 1).

## Установка

Требуется Python 3.10+.

```bash
cd survey_weighting_app
python -m venv .venv
```

Активируйте виртуальное окружение (Windows PowerShell):

```powershell
.\.venv\Scripts\Activate.ps1
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

Браузер откроет интерфейс приложения.

## Формат данных

**Важно:** названия колонок в ваших Excel могут быть любыми. Сервис не привязан к фиксированным именам: вы сами указываете, какая колонка опроса соответствует какой переменной в файле Росстата (или строкам/столбцам матрицы).

- **Строка заголовков** задаётся в боковой панели (по умолчанию вторая строка листа = типичная подпись переменных).
- Файл **Росстата** может быть:
  - **матрицей**: первый столбец — подписи строк (например, пол×возраст), остальные столбцы — регионы/типы городов, в ячейках — численность;
  - **таблицей**: столбцы — измерения + столбец с численностью или долей (автоопределение).

В папке `examples/` должны лежать два простых Excel для проверки интерфейса. Если их нет, выполните из корня проекта:

`python build_examples.py`

Файлы: `survey_example.xlsx` (опрос) и `rosstat_targets_example.xlsx` (матрица целей). В примере заголовки колонок — в **первой** строке листа (в боковой панели укажите номер строки 1).

## Режимы

1. **Внешние цели (Росстат)** — подгонка распределения выборки под целевые доли. Одна переменная: вес ∝ целевая доля / фактическая. Несколько переменных: **IPF** (до 50 итераций, допуск 0.0001). Можно исключить ненужные группы из целей до расчёта.
2. **Группа на группу** — одна переменная делит выборку на две группы; соц-дем структура взвешиваемой группы выравнивается под целевую. Пропуски в переменной-разделителе можно выбрать как отдельную категорию «— Пропуск —».

## Предпросмотр и объединение страт

После расчёта показывается компактная таблица по стратам. **Сырой вес** &lt; 0.3 подсвечивается розовым, &gt; 2.5 — оранжевым. Для экстремальных страт под таблицей можно выбрать страту для **объединения** (в рамках сессии), затем нажать «Применить объединение» и пересчитать.

Интерактивное выделение ячеек мышью в Streamlit ограничено; объединение реализовано через выпадающие списки по каждой экстремальной страте.

## Сброс

Кнопка **«Сбросить всё»** очищает сессию (включая загруженный файл Росстата). Файл опроса нужно загрузить снова.
