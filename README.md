# Подгонка выборки к целям (Росстат)

Веб-приложение на **Streamlit**: постстратификация опроса по внешним целям (матрица или таблица с численностями). В колонку **WT** записываются веса без лишних шагов.

## Логика

- В каждой **ячейке** (пересечение выбранных признаков): вес = целевая доля ÷ фактическая доля в выборке.
- **Взвешенное** распределение по этим ячейкам совпадает с целевым (с учётом ваших исключений в файле Росстата).
- Отдельная нормировка «средний вес = 1» **не выполняется** (она только масштабировала бы все веса и не меняла бы подгонку). При полном покрытии ячеек и сумме целевых долей, равной 1, **сумма весов равна числу респондентов N**.

## Развёртывание на Streamlit Community Cloud

1. Репозиторий на GitHub с `app.py` и `requirements.txt`.
2. [share.streamlit.io](https://share.streamlit.io) → подключить репозиторий.
3. **Main file path:** `app.py`.

Локально: Python **3.10+**.

## Установка и запуск

```powershell
cd survey_weighting_app
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

## Данные

- Строка с именами переменных в опросе задаётся в боковой панели (часто это **2-я** строка листа).
- Файл Росстата: **матрица** (строки × столбцы с численностями) или **таблица** с измерениями и столбцом численности/доли.
- При скачивании в исходный Excel добавляется столбец `WT` (верхние строки листа сохраняются при успешной записи через openpyxl).

## Примеры

`python build_examples.py` создаёт `examples/survey_example.xlsx` и `examples/rosstat_targets_example.xlsx` (для них в панели укажите строку заголовков **1**).
